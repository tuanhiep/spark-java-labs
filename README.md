# Spark Java Labs

Java learning labs for Apache Spark.

## Purpose

This repository keeps course-based Spark practice material in Java. It is a learning/reference repo, not a production data pipeline.

## Contents

- Java Spark examples.
- Local data-processing exercises.
- Generated output folders may appear during local runs and should not be treated as source.

## How To Use

Open the project in IntelliJ IDEA and run individual Spark examples from the IDE or command line.

## Status

Active learning repo. The next upgrade is to document prerequisites, sample input data, expected output, and cleanup commands for generated Spark artifacts.

## Validation

Run each Spark example against a known input and verify the output path, record count, and cleanup behavior.

## Engineering Notes

Generated Spark output should not be treated as source. Keep examples small, repeatable, and easy to reset.
